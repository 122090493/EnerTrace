# EnerTrace 能量追踪系统 ⚡️

> 一个基于 React + TypeScript 的能量消耗可视化与管理系统。

## 📖 项目简介
本项目旨在提供一个直观的能量追踪解决方案。它已经配置为可导出为静态站点，支持从日常记录到月度报表的全方位数据查看。

## ✨ 核心功能
- **Daily (每日概览)**: 实时查看当天的能量消耗趋势。
- **Weekly (周报分析)**: 按周统计，分析短期波动。
- **Monthly (月度总结)**: 长周期数据聚合，发现长期规律。
- **Report (详细报表)**: 生成可视化的数据图表。
- **Settings (个性化设置)**: 自定义追踪目标与偏好。

## 🛠️ 技术栈
- **Frontend**: React, TypeScript
- **Build Tool**: Vite / Webpack (根据实际调整)
- **Deployment**: Netlify (已配置 `netlify.toml`)
